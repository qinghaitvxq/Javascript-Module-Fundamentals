### Why Modules Matters In Javascript?
*模块是什么？*
> 模块是一组有特定功能的代码和数据，它封装了一些实现细节，暴露出公共的API，通过和其他模块的连接来打造出大型应用。

*模块化的目的*
* create higher-level abstractions
* Encapsulation
* Reusability
* Simplify dependency management
